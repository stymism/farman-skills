==========================================================
 Plaud-HTML スキル ポータブルパック（別PC用）
==========================================================

このZIPは「別PC（別OneDriveアカウント）」で /plaud-html を
使えるようにするための一式です。

----------------------------------------------------------
■ 使い方（カンタン3ステップ）
----------------------------------------------------------
1) このZIPを別PCの好きな場所に展開（解凍）する
2) 展開フォルダ内の「INSTALL.cmd」をダブルクリックする
3) 完了したら Claude Code を再起動し、 /plaud-html を実行

これだけで以下が自動配置されます：
  - スキル本体   → ~/.claude/skills/plaud-html
                   （CLAUDE_CONFIG_DIR 設定PCではそのフォルダ配下）
  - 設定ファイル → ~/.plaud/plaud-config.json（APIトークン入り）
  - 作業データ   → そのPCの OneDrive（無ければ Documents）\plaud_summaries
    ※ work_dir はそのPCの実パスへ自動で書き換わります

----------------------------------------------------------
■ 前提（これだけ確認）
----------------------------------------------------------
・別PCの Claude Code に「同じ Claude アカウント」でログインしていること
  → Plaud / Asana の連携（MCPコネクタ）はアカウントに付いてくるので
    トークン設定は不要です。
・デプロイ（Cloudflare）まで使う場合のみ、未導入なら:
    npm install -g wrangler

----------------------------------------------------------
■ 中身
----------------------------------------------------------
  INSTALL.cmd ............ これをダブルクリック
  INSTALL.ps1 ............ 実体（cmdから呼ばれる）
  skill\ ................. スキル本体一式（SKILL.md ほか）
  plaud-config.json ...... 設定（Plaud/Cloudflareトークン入り）
  plaud_summaries\ ....... 既存サイトデータ（HTML・index・functions）
                           ※ node_modules は除外済み（不要）

----------------------------------------------------------
■ 注意
----------------------------------------------------------
・plaud-config.json には APIトークンが含まれます。取り扱い注意。
・2台のPCは OneDrive が別アカウントのため、生成物は自動同期されません。
  各PCはそれぞれローカルで動作し、Cloudflare へは同じサイトにデプロイされます。
